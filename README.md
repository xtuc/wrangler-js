# wrangler-js

> Wrangle your Cloudflare workers - JavaScript edition.

## Install

Using npm:

```sh
npm install --save wrangler-js
```

or using yarn:

```sh
yarn add wrangler-js
```

## Usage

The following env variables must be defined:
- `CF_ACCOUNT_ID`
- `CF_API_KEY`
- `CF_ACCOUNT_EMAIL`

cli:

```js
$ wrangler-js
```
